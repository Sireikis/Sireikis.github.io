Meals International

My most recent app, Meals International is a basic food recipes app. Information is obtained from TheMealsDB API and displayed on a table view. Each cell leads to a view displaying more detailed information about the recipe -- which is what is seen in the displayed image on this page. Utilizes Combine for handling API requests and UI updates, Core Data for persistence, and UIKit for the UI. Take a look at the Meals International GitHub for a more detailed writeup!

ADD - Algorithms, Design Patterns, Data Structures

ADD is a reference handbook for fundamental concepts in computer science. This app helped motivate me to learn about concepts that a self-taught developer might otherwise lack. The architecture of this app was constructed utilizing Bridge, Adapter, and Proxy patterns to allow for lazy instantiation of views and flexible content. Admittedly the application of a more standard overarching design pattern and separation into appropriate layers would likely result in a cleaner app. The content is currently hosted locally on a user's device in static web pages that are displayed using WKWebView. Syntax highlighting of code is achieved using PrismJS
